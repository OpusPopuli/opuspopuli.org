# Brand Assets

This directory contains all Opus Populi brand assets.

**For complete brand guidelines, see [`BRAND.md`](../BRAND.md) in the repository root.**

## Contents

- `favicons/` - Browser icons (16px, 32px, 512px)
- `logos/` - Logo files (SVG and PNG)
- `social/` - Social media banners
- `index.html` - Interactive style guide (open in browser)
